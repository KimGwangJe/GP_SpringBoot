### 📢 프로젝트 소개
이 프로젝트는 호서대학교 학생들을 위한 챗봇 서비스로 사용자가 메시지를 보내면 웹서버를 통해 DialogFlow가 질문의 의도를 분석하고 질문이 정의된 의도(intent)에 해당하면 Django API로 요청을 보내 적합한 답변을 생성하고 정의 되지 않은 질문에 대한 답변은 파인튜닝된 ChatGPT 모델을 통해 제공합니다.

챗봇은 다음과 같은 질문의 의도를 지원합니다.

1️⃣ 사용자가 수강한 강의 기반 강의 추천 : 사용자가 수강한 강의 데이터를 기반으로 맞춤 강의를 추천합니다.

2️⃣ 질문에 기반한 강의 추천 : 사용자가 질문에 입력한 키워드를 추출하여 강의를 추천합니다.

3️⃣ 졸업 요건 확인 : 졸업 요건을 분석해 개인 맞춤형 커리큘럼을 제안합니다.

4️⃣ 학교 시설 안내 : 캠퍼스 내 시설 위치 정보를 제공합니다.

### 🙋‍♂️ 역할
프로젝트에서 저는 Spring Security와 JWT를 이용해 로그인 기능을 구현하고 토큰 기반 인증 방식을 통해 세션 기반 인증보다 안전한 인증 메커니즘을 제공했습니다. 이를 통해 사용자 인증 처리의 효율성을 높이고 보안을 강화할 수 있었습니다.

ChatGPT API를 연동하여 정의되지 않은 질문에 대한 답변을 생성하는 기능을 추가함으로써 보다 향상된 사용자 경험을 제공했습니다. 이러한 기능 통합을 통해 서비스의 질을 높여 경쟁력을 강화할 수 있었습니다.

애플리케이션의 성능 최적화도 중요한 목표였습니다. Redis를 활용해 GET 요청의 결과를 캐싱함으로써 자주 조회되는 데이터에 대한 응답 시간을 획기적으로 줄였습니다. JMeter로 부하 테스트를 수행한 결과 다수의 요청이 들어왔을 때 응답 시간이 평균 3200ms에서 470ms로 단축되었고 데이터베이스의 부하도 감소 되었습니다. 이를 통해 시스템의 전반적인 성능을 크게 향상시켰습니다.

또한, 애플리케이션은 Docker를 사용해 컨테이너화하여 배포 과정을 단순화하고 다양한 환경에서의 애플리케이션 실행을 용이하게 했습니다. 최종적으로 애플리케이션을 AWS에 배포하여 클라우드 환경에서 확장성과 가용성을 확보하고 실제 운영 환경에서도 안정적인 성능을 제공할 수 있도록 했습니다.

이러한 노력의 결과로 2024년 호서대학교 컴퓨터공학부 졸업작품 발표회에서 최우수상을 수상하며 프로젝트의 성과를 인정받았습니다.
